云台任务介绍

云台任务

主要文件

机器人的云台任务涉及到相当多的头文件及源文件，较为重要的源文件包括：

  • gimbal_task.c
    执行云台完整任务的源文件，在操作系统文件freertos.c中被调用。

  • gimbal_behaviour.c
    处理云台姿态和遥控器控制相关行为的源文件，在gimbal_task.c中被调用。

  • shoot.c
    执行云台设计功能的源文件，在gimbal_task.c中被调用。这部分内容的介绍包含在 servo_task 中

  • CAN_receive.c
    CAN中断接收文件，在云台任务中主要用于向云台yaw轴电机，pitch轴电机，拨弹电机等发送控制电流，在gimbal_task.c中被调用。

存储类型及指针类型

gimbal_control_t:

typedef struct
{
    const RC_ctrl_t *gimbal_rc_ctrl;
    const fp32 *gimbal_INT_angle_point;  
    const fp32 *gimbal_INT_gyro_point;  
    gimbal_motor_t gimbal_yaw_motor;  
    gimbal_motor_t gimbal_pitch_motor;  
    gimbal_step_cali_t gimbal_cali;  
} gimbal_control_t;

gimbal_control_t 类型包含了云台的所有信息，包括：遥控器接收数据存和yaw轴，pitch轴的电机数据。

gimbal_motor_t:

typedef struct
{
    const motor_measure_t *gimbal_motor_measure;
    gimbal_PID_t gimbal_motor_absolute_angle_pid;
    gimbal_PID_t gimbal_motor_relative_angle_pid;
    pid_type_def gimbal_motor_gyro_pid;
    gimbal_motor_mode_e gimbal_motor_mode;
    gimbal_motor_mode_e last_gimbal_motor_mode;
    uint16_t offset_ecd;
    fp32 max_relative_angle; //rad
    fp32 min_relative_angle; //rad

    fp32 relative_angle;     //rad
    fp32 relative_angle_set; //rad
    fp32 absolute_angle;     //rad
    fp32 absolute_angle_set; //rad
    fp32 motor_gyro;         //rad/s
    fp32 motor_gyro_set;
    fp32 motor_speed;
    fp32 raw_cmd_current;
    fp32 current_set;
    int16_t given_current;

} gimbal_motor_t;

gimbal_motor_t 类型定义了单个云台电机的数据存储结构。其中，遥控器设置模式存储于gimbal_motor_mode_e枚举类型的结构中。

gimbal_step_cali_t:

gimbal_step_cali_t 类型定义了云台电机角度及其编码值的最大最小值，用以校准云台电机中值。

云台任务工作流程

gimbal_task:

gimbal_init(&gimbal_control)

  • PID初始化。
  • 获取指向电机数据，陀螺仪数据，遥控器数据的指针。
  • 初始化云台电机模式为 GIMBAL_MOTOR_RAW 状态。

shoot_init()

对拨弹轮电机数据进行初始化

判断电机是否在线

通过函数 toe_is_error(uint8_t toe) 判断yaw轴和pitch轴电机有无掉线，若有电机掉线，则阻塞云台程序运行。

gimbal_set_mode(&gimbal_control)

设置云台控制模式，调用函数void gimbal_behaviour_mode_set(gimbal_control_t *gimbal_mode_set)，该函数在源文件gimbal_behaviour.c中被定义。这个函数有点长，至下文的gimbal_behaviour.c部分查看。

gimbal_mode_change_control_transit(&gimbal_control)

控制云台模式切换，若发生电机模式切换，则重新赋值电机状态。
云台状态摘要：
typedef enum
{
GIMBAL_MOTOR_RAW = 0,
GIMBAL_MOTOR_GYRO,
GIMBAL_MOTOR_ENCONDE,
} gimbal_motor_mode_e;
云台状态数值为枚举类型，被定义于头文件gimbal_task.h中，分别为：
1.电机原始值控制
2.电机（C板）陀螺仪角度控制
3.电机编码值角度控制

gimbal_feedback_update(&gimbal_control)

该函数进行云台当前数据的反馈，函数检验完参数是否是空指针后，完成以下任务：

  • 利用指针获取yaw轴，pitch轴电机绝对角度的更新值。
  • 通过云台电机自身编码值和电机转动半径的关系，解算出电机和底盘的相对角度。

gimbal_set_control(&gimbal_control)

该函数设置云台控制相关数值，完成以下任务：

  • 云台电机增减角度赋默认值（0.0f）
  • 调用函数 gimbal_behaviour_control_set(&add_yaw_angle, &add_pitch_angle, set_control),该函数定义在源文件gimbal_behaviour.c中。该函数检验完参数是否是空指针后，根据设置的云台行为模式调用各自的设置函数，具体执行内容于gimbal_behaviour.c源文件中解释。

gimbal_control_loop(&gimbal_control)

该函数计算云台控制PID，函数检验完参数是否是空指针后，完成以下任务：

  • 根据云台电机状态（GIMBAL_MOTOR_RAW，GIMBAL_MOTOR_GYRO，GIMBAL_MOTOR_ENCONDE），调用对应模式的pid调试函数，这些函数定义于源文件gimbal_task.c中。
  • 对角度环，速度环进行串级pid调试。
  • 赋值电流值。

shoot_can_set_current = shoot_control_loop()

云台控制射击任务，其中函数 shoot_control_loop() 返回控制拨弹电机电流值，该函数被定义于源文件shoot.c中。

CAN_cmd_gimbal(yaw_can_set_current, pitch_can_set_current, shoot_can_set_current, 0)

该函数定义于源文件 CAN_receive.c 中，和底盘任务中通过CAN线发送封包的函数功能相同，利用HAL库封装好的函数向CAN总线发送封包。其中，封包的首地址为0x1FF。

CAN_GIMBAL_ALL_ID = 0x1FF

源文件 gimbal_behaviour.c

云台任务中的部分函数被定义在源文件gimbal_behaviour.c中，该文件的作用为辅助源文件 gimbal_task.c 完成云台控制任务，提供姿态解算函数等。该文件不由操作系统文件freertos.c直接调用，只由 gimbal_task() 函数调用，起到连接底层函数与表层函数的作用。

云台行为状态机

云台行为状态被对应为一个枚举类型，该枚举类型被定义于头文件 gimbal_behaviour.h 中。

typedef enum
{
    GIMBAL_ZERO_FORCE = 0, 
    GIMBAL_INIT,           
    GIMBAL_CALI,           
    GIMBAL_ABSOLUTE_ANGLE, 
    GIMBAL_RELATIVE_ANGLE, 
    GIMBAL_MOTIONLESS,     
} gimbal_behaviour_e;
  • 根据当前云台行为模式的不同，函数 void gimbal_behaviour_mode_set(gimbal_control_t *gimbal_mode_set) 中设置的云台电机状态机模式也会不同。
  • 根据当前云台行为模式的不同，函数 void gimbal_control_loop(gimbal_control_t *control_loop) 会调用不同的云台控制函数。

云台行为控制

任务中通过调用 gimbal_behaviour_control_set(&add_yaw_angle, &add_pitch_angle, set_control) ,引入yaw轴，pitch轴电机的增量角，并根据云台电机状态机的模式，向电机发送控制值。该函数被 void gimbal_set_control(gimbal_control_t *set_control) 调用，但其本身定义于源文件gimbal_behaviour.c中。
下列函数均定义于源文件gimbal_behaviour.c中，且被云台行为控制函数调用：

下列函数均为接受3个参数的无返回值函数：

fp32 *yaw: yaw轴电机角度增量，为上级函数中定义的局部变量的指针。
fp32 *pitch: pitch轴电机角度增量，为上级函数中定义的局部变量的指针。
gimbal_control_t *gimbal_control_set: 云台控制总数据结构，为全局变量的指针。

云台无力模式

gimbal_zero_force_control(add_yaw, add_pitch, gimbal_control_set);

当云台行为模式是 GIMBAL_ZERO_FORCE 时，这个函数会被调用。函数检验完参数是否包含空指针后，将角度增量值置零。 

*yaw = 0.0f;
*pitch = 0.0f;
云台初始化控制模式

gimbal_init_control(add_yaw, add_pitch, gimbal_control_set);

当云台行为模式是 GIMBAL_INIT 时，这个函数会被调用。函数检验完参数是否包含空指针后，判断pitch轴电机当前角度与预设角度的差值是否大于误差阈值，再根据电机当前角度与初始预设角度的差值，计算两轴电机角度增量，使电机在限定时间内完成姿态初始化。

   if (fabs(INIT_PITCH_SET - gimbal_control_set->gimbal_pitch_motor.absolute_angle) > GIMBAL_INIT_ANGLE_ERROR)
{
    *pitch = (INIT_PITCH_SET - gimbal_control_set->gimbal_pitch_motor.absolute_angle) * GIMBAL_INIT_PITCH_SPEED;
    *yaw = 0.0f;
}
else
{
    *pitch = (INIT_PITCH_SET - gimbal_control_set->gimbal_pitch_motor.absolute_angle) * GIMBAL_INIT_PITCH_SPEED;
    *yaw = (INIT_YAW_SET - gimbal_control_set->gimbal_yaw_motor.relative_angle) * GIMBAL_INIT_YAW_SPEED;
}
云台校准控制模式

gimbal_cali_control(add_yaw, add_pitch, gimbal_control_set);

当云台行为模式是 GIMBAL_CALI 时，这个函数会被调用。函数检验完参数是否包含空指针后，分别获取yaw轴，pitch轴电机旋转角度的最大和最小值。

云台陀螺仪控制模式

gimbal_absolute_angle_control(add_yaw, add_pitch, gimbal_control_set);

当云台行为模式是 GIMBAL_ABSOLUTE_ANGLE 时，这个函数会被调用。函数检验完参数是否包含空指针后，基于从遥控器接收到的对应通道的数值与鼠标点击值的大小，计算yaw轴，pitch轴旋转角度增量。
该模式有通过键盘控制云台180°转动的功能。

static int16_t yaw_channel = 0, pitch_channel = 0;

rc_deadband_limit(gimbal_control_set->gimbal_rc_ctrl->rc.ch[YAW_CHANNEL], yaw_channel, RC_DEADBAND);
rc_deadband_limit(gimbal_control_set->gimbal_rc_ctrl->rc.ch[PITCH_CHANNEL], pitch_channel, RC_DEADBAND);

*yaw = yaw_channel * YAW_RC_SEN - gimbal_control_set->gimbal_rc_ctrl->mouse.x * YAW_MOUSE_SEN;
*pitch = pitch_channel * PITCH_RC_SEN + gimbal_control_set->gimbal_rc_ctrl->mouse.y * PITCH_MOUSE_SEN;
云台编码值控制模式

gimbal_relative_angle_control(add_yaw, add_pitch, gimbal_control_set);

当云台行为模式是 GIMBAL_RELATIVE_ANGLE 时，这个函数会被调用。函数检验完参数是否包含空指针后，基于从遥控器接收到的对应通道的数值与鼠标点击值的大小，计算yaw轴，pitch轴旋转角度增量。

基本代码与陀螺仪控制模式一致。

云台遥控器无输入控制模式

gimbal_motionless_control(add_yaw, add_pitch, gimbal_control_set);

当云台行为模式是 GIMBAL_MOTIONLESS 时，这个函数会被调用。函数检验完参数是否包含空指针后，将角度增量值置为零。
（与云台无力模式相同）
