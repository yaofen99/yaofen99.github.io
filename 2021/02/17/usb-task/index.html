USB 模块介绍

本模块最重要的功能是定义了 usb_printf()函数，提供了usb模块出现异常时的处理方式，同时也事无巨细得罗列出了usb工作的各种状态。由于USB任务相对底层，所以在很多地方直接使用了全局变量。

常用指针和变量

直接看名字就大概知道这些指针是用来干啥的了

  • USBD_HandleTypeDef hUsbDeviceFS
    定义了各种USB的id，工作状态，及一些函数指针
  • USBD_DescriptorsTypeDef FS_Desc
  • USBD_ClassTypeDef USBD_CDC
  • USBD_CDC_ItfTypeDef USBD_Interface_fops_FS

工作流程

  • 此处的初始化直接检验了几个全局变量（FS_Desc，USBD_CDC， USBD_Interface_fops_FS， ）的句柄是否存在（句柄的处理可追溯到HAL库的操作）。但是这里的错误处理函数留空，需要定制。
  • 默认输出错误信息

函数

static void usb_printf(const char *fmt,…)

标准的USB输出函数，可直接调用。
